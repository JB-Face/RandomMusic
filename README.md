# MUSIC TRIP

随着现在推荐算法越来越厉害，大家听的音乐越来越圈子化。然而这世界还有很多很多音乐可以欣赏
就像nujabes 一样发掘不同的音乐让大家感受到

# 声明
本软件依附于网易云，本软件本身无法提供任何音乐
